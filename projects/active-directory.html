Active Directory & Domain Services Lab

Windows Server · Active Directory Domain Services · DNS · Group Policy · User & Access Management

Overview

Built and managed a single-domain Windows Server environment to practise enterprise identity and network administration. The lab focused on centralised authentication, user and group management, policy enforcement, and secure access to network resources.

Objectives

  • Deploy and configure a Windows Server domain controller.
  • Implement Active Directory Domain Services (AD DS) for centralised identity management.
  • Configure DNS to support domain authentication and name resolution.
  • Manage users, groups, and organisational units (OUs).
  • Apply Group Policy to enforce security and configuration standards.

Lab Setup

  • Windows Server installed and promoted to a domain controller.
  • Active Directory Domain Services and DNS roles configured.
  • Client machines joined to the domain for authentication testing.
  • Structured OU hierarchy to support administrative control and policy application.

What I Did

  • Created and managed domain user accounts and security groups.
  • Organised users and computers into OUs based on administrative needs.
  • Configured Group Policy Objects (GPOs) to enforce password, security, and system settings.
  • Joined client systems to the domain and validated domain-based authentication.
  • Administered access to resources using group-based permissions.

What I Tested and Validated

  • Authentication: confirmed successful domain logons from client machines.
  • Policy enforcement: validated GPO application on domain-joined systems.
  • Access control: verified permissions were applied correctly via group membership.
  • Directory health: confirmed stable AD and DNS operation after configuration changes.

Key Outcomes

  • Developed practical experience managing enterprise identity and access systems.
  • Strengthened understanding of how authentication, authorisation, and policy enforcement work together.
  • Built confidence administering Windows Server environments in a structured, repeatable way.

Tools

  • Windows Server
  • Active Directory Domain Services (AD DS)
  • DNS
  • Group Policy Management
  • Domain-joined client systems
